# 제주 TOU 공통고객 2개년 종단분석·요금 모니터링·제어 시뮬레이터 v13

2024년과 2025년 고객번호가 일치하고 양 연도 전체기간 및 시간값 완전도 기준을 충족한 712명을 대상으로 분석합니다.

## v13 추가 기능
- 고객별 요금 모니터링: 연간 전체 또는 월중 조회일 기준
- 일반 주택용·제주 TOU·구독 기본형·구독 프리미엄형 요금 비교
- 고객별 추천요금제, 제공량 사용률, 알림단계, 예상 절감액
- 고객별 종단진단 및 목표관리·행동권고·한전 직접제어 위임
- 전월 요금, 전년 동월 요금, 제공량, 직접 입력 목표요금 기준 감축대안
- 주중·주말 관리 전후 예상 부하곡선
- 3~8개 공통 군집과 2024→2025 군집 전이
- 실제 고객 100명 포트폴리오와 변압기 가상 운전한도 최적화
- 표의 연속형 수치는 소수점 첫째자리, 비중·증감률·오차·사용률은 % 표시

## 실행
```bash
pip install -r requirements.txt
streamlit run streamlit_app_actual_tou_v13.py
```

## 주의사항
- 가전별 절감·제어량은 단일 계량값에서 직접 식별한 값이 아니라 등록기기와 표준 절감 라이브러리를 이용한 시나리오입니다.
- 요금은 2026년 6월 시행 요금표를 2024·2025년 사용량에 동일 적용한 비교입니다.
- 100가구는 실제 동일 변압기 연결고객이 아니며, 변압기 한도는 제어 전 최대부하 대비 가상 비율입니다.
