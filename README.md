# 2025 제주 TOU 실제고객 기반 구독형 요금·수요관리 시뮬레이터 v8

## 1. 목적
기존 시뮬레이터의 가상 100가구를 2025년 제주 주택용 TOU 실제 시간대별 계량자료의 비식별 핵심표본 914명으로 교체한 버전입니다.

## 2. 반영된 실제 데이터
- 원자료 고객: 1,210명
- 핵심표본: 914명
- 핵심표본 기준: 365일 자료 보유 및 시간값 완전도 99.5% 이상
- 시간해상도: 1시간
- 기간: 2025년 1월 1일~12월 31일
- 고객번호: 제거 후 T0001 형식으로 재부여

## 3. 주요 기능
1. 실제 고객 사용패턴 군집 및 전체 현황
2. 고객별 월 중간 시점의 월말 사용량·요금 예측
3. 일반 주택용·제주 TOU·구독 기본형·프리미엄형 요금 비교
4. 남은 정액 사용량, 알림 단계, 목표 감축량 계산
5. 등록 가전정보 기반 행동권고 조합
6. 한전 직접제어 위임 시 예상 감축·시간이동 계획
7. 주중·주말 평균 시간대 부하곡선의 관리 전후 비교
8. 실제 고객 100명 표본의 요금제 추천과 포트폴리오 부하곡선

## 4. 요금 가정
- 기본형: 84,900원 / 450kWh
- 프리미엄형: 249,900원 / 1,000kWh
- 초과단가: 기본 300원/kWh, 화면에서 변경 가능
- 현행 요금: 2026년 6월 1일 시행 요금표 적용
- 제주 TOU 계약전력: 기본 3kW 가정
- 부가가치세 10%, 전력산업기반기금 2.7% 반영
- 기후환경요금 및 연료비조정요금: 적용단가 미확인으로 제외

따라서 요금 결과는 2025년 실제 청구액 재현이 아니라, 2025년 사용량에 2026년 요금표를 적용한 비교 시뮬레이션입니다.

## 5. 파일 구성
- `streamlit_app_actual_tou_v8.py`: 실행파일
- `tou_v8_summary.json`: 전체 요약·군집 정보
- `tou_v8_customer_metrics.csv`: 고객별 비식별 지표
- `tou_v8_daily_usage.csv.gz`: 고객별 일사용량
- `tou_v8_customer_profiles.csv.gz`: 계절·주중/주말별 24시간 평균곡선
- `tou_v8_monthly_tou.csv.gz`: 고객별 월간 TOU 시간대 사용량과 요금
- `requirements.txt`: 라이브러리 목록

## 6. 실행
```bash
pip install -r requirements.txt
streamlit run streamlit_app_actual_tou_v8.py
```

## 7. 해석상 주의
- 전체 계량값으로 특정 가전의 실제 작동을 확정하지 않습니다.
- 가전별 감축량은 고객이 등록한 기기와 최초 표준 가전자료를 이용한 계획값입니다.
- 직접제어에는 고객 동의, 스마트가전·스마트플러그·HEMS 연계가 필요합니다.
- 시간이동은 피크를 낮출 수 있지만 월간 kWh 감축에는 포함되지 않습니다.

## 8. 보안
비식별 ID를 사용하더라도 고객별 시간대 사용패턴은 민감한 정보입니다. 공개 GitHub 또는 공개 Streamlit Community Cloud에 데이터 파일을 업로드하지 말고, 한전 내부망 또는 접근통제 서버에서 사용해야 합니다.
