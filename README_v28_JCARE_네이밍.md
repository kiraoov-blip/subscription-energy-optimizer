# J-CARE v28

## 공식 명칭
- 국문: 제주 TOU 고객 맞춤형 요금·에너지관리 시뮬레이터
- 영문: Jeju Customer Analytics, Rate Recommendation & Energy Management Simulator
- 약칭: J-CARE

## 화면 변경사항
- 첫 화면 상단에 `J-CARE` 브랜드명 표시
- 국문 공식명과 영문 공식명을 함께 표시
- 웹 브라우저 상단 탭 제목은 `J-CARE`로만 표시
- 기존 분석·요금·제어 기능 및 데이터 구조는 유지

## 호환성
v27과 데이터 구조가 동일하므로 기존 저장소에는 실행파일만 추가하면 됩니다.
