# 제주 TOU 공통고객 2개년 종단분석·수요관리 시뮬레이터 v12

## 분석대상
- 2024년과 2025년 고객번호가 일치하는 고객 중 712명
- 2024년 366일, 2025년 365일 자료 보유
- 양 연도 시간대별 계량값 완전도 99.5% 이상
- 원 고객번호는 제거하고 M0001 형식으로 비식별화

## 주요 기능
1. 2024·2025 월별·계절별 사용량과 TOU 시간대 비중 비교
2. 고객별 사용량·부하곡선·요금제·예측성능 종단진단
3. 공통 군집 3~8개 설정 및 군집 전이행렬
4. 패턴 안정성점수·수요관리 우선점수·구조변화 신호
5. 연간·월별 추천요금제 유지·변경 분석
6. 2024년 패턴을 활용한 2025년 월말 예측 검증
7. 동일 고객 100명 포트폴리오의 2개년 부하 비교
8. 변압기 목표 운전한도와 직접제어 참여율에 따른 피크관리 시뮬레이션

## 실행
```bash
pip install -r requirements.txt
streamlit run streamlit_app_actual_tou_v12.py
```

## 주의사항
- 요금은 2026년 6월 시행 요금표를 과거 사용량에 동일 적용한 시뮬레이션입니다.
- 변압기 운전한도는 실제 설비 정격이 아니라 선택 고객군 최대부하의 일정 비율입니다.
- 고객별 가전 사용량과 실제 스마트가전 제어 가능량은 포함되지 않습니다.
